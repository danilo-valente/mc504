QOTD File System
================

### Requirements

 - gcc
 - libfuse

### Build

```
make build
```

### Mount

```
make mount
```

### Unmount

```
make umount
```

### Testing

After mounting, open another terminal window, navigate to ./mnt and read the quote file contents to get your quote-of-the-day.
